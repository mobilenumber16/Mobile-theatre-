Cloudflare Stream Uploader
===========================

This project uploads videos to Cloudflare Stream using API.

Setup:
1. Go to https://dash.cloudflare.com/
2. Create API Token with Stream access
3. Replace placeholders in .env with:
   CLOUDFLARE_ACCOUNT_ID
   CLOUDFLARE_API_TOKEN

To run:
-----------
npm install
node index.js
